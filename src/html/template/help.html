FAQ

  • What is this for?
  • I wanted to keep track of my spendings, see what i'm using money on and optionally create different statistics about it, see trends about my savings.
  • Is my data safe/stored?
  • This webapp uses your browsers localStorage to store data, which means, it never leaves your actual phone/computer. However keep in mind, if you use "Delete browser history" or anything like that, your data is lost. Make sure to backup your data by saving (copy into notepad) the serialized format (which you can import later) on the configure page!
